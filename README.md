# iqbaltestgithub

Integration test between GitHub and Jira

## Description

This repository is designed for testing the integration between GitHub and Jira. It serves as a testing ground for various integration features and workflows.

## Features

- GitHub and Jira integration testing
- Simple HTML interface
- Documentation and examples

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/kebalicious/iqbaltestgithub.git
   ```

2. Open the `index.html` file in your web browser to view the simple HTML page.
